Description
In this project you can see the relationship of the survival rate of different sex, class, and age.
This was programmed using R in R Studio

The Titanic accident was horrible, however based on the data we can now make predictions.
We can predict that if you are a female you had a very high survival rate and would even increase further if you were young.
However just because you are a female doesnt mean you're guaranteed to survive.
After all, if you aren't wealthy and buy a 3rd class ticket, you have a very low chance to survive.
If you are a man in 2nd or 3rd class, you would be very lucky survive.
You can also see in the graphs that most if not all children survived

They put children and women first. Making sure the children can have a future and the women can birth more.

In the code there is a short description of each visual.
The final visual is my favorite and can tell us the most about the data.

Code

options(digits = 3) # report 3 significant digits
library(tidyverse)
library(titanic)
library(ggplot2)
library(dplyr)

titanic <- titanic_train %>%
select(Survived, Pclass, Sex, Age, SibSp, Parch, Fare) %>%
mutate(Survived = factor(Survived),
Pclass = factor(Pclass),
Sex = factor(Sex))

# plot 1 - survival filled by sex with position_dodge
titanic %>%
ggplot(aes(Survived, fill = Sex)) +
geom_bar(position = position_dodge())

#plot 2 - sex filled by survival
titanic %>%
ggplot(aes(Sex, fill = Survived)) +
geom_bar()

#plot 3 - plcass filled by survival with position_fill
titanic %>%
ggplot(aes(Pclass, fill = Survived)) +
geom_bar(position = position_fill()) +
ylab("Proportion")

#plot 4 - survival based on sex, class, and age
titanic %>%
ggplot(aes(Age, y = ..count.., fill = Survived)) +
geom_density(alpha = 0.2) +
facet_grid(Sex ~ Pclass)
